Features

Interactive GUI
Both Server and Client GUIs include:
Scrollable chat window
Improved spacing and modern layout
File selection buttons
Status updates and connection notifications
File Transfer
Send files from client to server and server to client
Any file type supported (PDF, ZIP, images, videos, etc.)
Automatic saving of received files
Messages added to chat showing file name and time
